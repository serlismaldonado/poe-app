## Requisitos del sistema

### macOS
- macOS 10.13 o superior
- Procesador Apple Silicon (M1/M2/M3) o Intel

### Windows
- Windows 10 o superior
- 64-bit

### Linux
- Kernel 3.2+ con glibc 2.17+
- 64-bit

## Instalación

### macOS
1. Descarga el archivo `.dmg`
2. Ábrelo y arrastra `Poe.app` a la carpeta Aplicaciones
3. Ejecuta desde Launchpad o Spotlight

### Windows
1. Descarga el archivo `.msi`
2. Ejecuta el instalador y sigue las instrucciones

### Linux
- **Debian/Ubuntu**: `sudo dpkg -i Poe_1.0.0_amd64.deb`
- **Fedora/RHEL**: `sudo rpm -i Poe_1.0.0_x86_64.rpm`
- **AppImage**: `chmod +x Poe_1.0.0_amd64.AppImage && ./Poe_1.0.0_amd64.AppImage`
